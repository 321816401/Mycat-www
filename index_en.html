What is MyCAT?

  • Completely open source database cluster, it can replace expensive Oracle cluster.
  • It combines traditional database functionality, and distributed database capabilities.
  • Stand-alone database applications can be moved to the "cloud", without changing any code.
  • The core function is database sharding, maximum support 100 billion rows sharding in one logic table. Database sharding provides many advantages, such as faster reads and writes to the database, improved search response, smaller table sizes, and distribution of tables based on need.
  • The application can use it as a database agents, its backend uses the native MySQL protocol with a number of MySQL servers to communicate, you can use the JDBC protocol to communicate with the majority of mainstream database, and also supports NoSQL databases such as the Mongodb. The future will support more database types.
  • So the Mycat is a BigSQL, the big data technology based the SQL database.
